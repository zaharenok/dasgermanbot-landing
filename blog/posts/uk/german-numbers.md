# Німецькі цифри: Повний довідник Zahlen для початківців

Ви вивчаєте німецьку. Вам потрібно щось порахувати.

Заминяєтеся. **Eins, zwei, drei...** Як продовжити? А як щодо більших чисел, цін або номерів телефонів?

Німецькі цифри (Zahlen) необхідні для повсякденного життя — від покупок до визначення часу.

**Рішення:** Оволодійте німецькими цифрами з чіткими патернами, порадами щодо вимови та практичним використанням.

## Цифри 0-10

**Базова лічба:**

1. **Null** (нуль)
   - **Вимова:** [нуль]
   - **Використання:** Нічого, нуль

2. **Eins** (один)
   - **Вимова:** [айнс]
   - **Примітка:** Втрачає 's' при зміні іменників

3. **Zwei** (два)
   - **Вимова:** [цвай]
   - **Використання:** Базова лічба

4. **Drei** (три)
   - **Вимова:** [драй]
   - **Використання:** Базова лічба

5. **Vier** (чотири)
   - **Вимова:** [фір]
   - **Використання:** Базова лічба

6. **Fünf** (п'ять)
   - **Вимова:** [фюнф]
   - **Примітка:** 'ü' як в "фюр"

7. **Sechs** (шість)
   - **Вимова:** [зекс]
   - **Примітка:** 'z' звучить як 'ц'

8. **Sieben** (сім)
   - **Вимова:** [зі:бн]
   - **Використання:** Базова лічба

9. **Acht** (вісім)
   - **Вимова:** [ахт]
   - **Використання:** Базова лічба

10. **Neun** (дев'ять)
    - **Вимова:** [нойн]
    - **Використання:** Базова лічба

11. **Zehn** (десять)
    - **Вимова:** [цейн]
    - **Використання:** База для чисел-підлітків

## Цифри 11-20

**Числа-підлітки:**

12. **Elf** (одинадцять)
    - **Паттерн:** Унікальне слово

13. **Zwölf** (дванадцять)
    - **Паттерн:** Унікальне слово

14. **Dreizehn** (тринадцять)
    - **Паттерн:** Drei + zehn (три + десять)

15. **Vierzehn** (чотирнадцять)
    - **Паттерн:** Vier + zehn (чотири + десять)

16. **Fünfzehn** (п'ятнадцять)
    - **Паттерн:** Fünf + zehn (п'ять + десять)

17. **Sechzehn** (шістнадцять)
    - **Паттерн:** Sech +zehn (шість + десять)
    - **Примітка:** Втрачає 's' від sechs

18. **Siebzehn** (сімнадцять)
    - **Паттерн:** Sieb + zehn (сім + десять)
    - **Примітка:** Втрачає 'en' від sieben

19. **Achtzehn** (вісімнадцять)
    - **Паттерн:** Acht + zehn (вісім + десять)

20. **Neunzehn** (дев'ятнадцять)
    - **Паттерн:** Neun + zehn (дев'ять + десять)

21. **Zwanzig** (двадцять)
    - **Паттерн:** База для десятків

## Десятки (20-100)

**Кратні десяти:**

22. **Dreißig** (тридцять)
    - **Паттерн:** Drei + ßig (три + -дцять)
    - **Вимова:** [драйсих]

23. **Vierzig** (сорок)
    - **Паттерн:** Vier + zig (чотири + -дцять)

24. **Fünfzig** (п'ятдесят)
    - **Паттерн:** Fünf + zig (п'ять + -дцять)

25. **Sechzig** (шістдесят)
    - **Паттерн:** Sech + zig (шість + -дцять)

26. **Siebzig** (сімдесят)
    - **Паттерн:** Sieb + zig (сім + -дцять)

27. **Achtzig** (вісімдесят)
    - **Паттерн:** Acht + zig (вісім + -дцять)

28. **Neunzig** (дев'яносто)
    - **Паттерн:** Neun + zig (дев'ять + -дцять)

29. **Hundert** (сто)
    - **Використання:** База для сотень

## Складні числа

**Як німці комбінують цифри:**

**Важливий паттерн:** Німці говорять одиниці ПЕРШИМИ, потім десятки.

30. **Einundzwanzig** (двадцять один)
    - **Дослівно:** Один-і-двадцять
    - **Паттерн:** Ein + und + zwanzig

31. **Zweiunddreißig** (тридцять два)
    - **Дослівно:** Два-і-тридцять
    - **Паттерн:** Zwei + und + dreißig

32. **Dreiundvierzig** (сорок три)
    - **Дослівно:** Три-і-сорок
    - **Паттерн:** Drei + und + vierzig

33. **Vierundfünfzig** (п'ятдесят чотири)
    - **Дослівно:** Чотири-і-п'ятдесят
    - **Паттерн:** Vier + und + fünfzig

## Культурні нотатки

### Цифри в німецькій культурі

- **Glückszahlen:** 7 (sieben) — щаслива, 13 (dreizehn) — нещаслива
- **Формат цін:** Німці використовують кому для десяткових (5,99 €)
- **Номери телефонів:** Вимовляються парами (zwei-vier-null, acht-zehn...)
- **Дати:** Формат день-місяць-рік (29.04.2026)
- **Нумерація поверхів:** Цокольний поверх = 0 (Erdgeschoss), перший поверх = 1 (erster Stock)

### Числові ідіоми

- **Alle neune** (все ідеально) — дослівно "всі дев'ятки"
- **Eine Nummer größer** (на розмір більше) — дослівно "число більше"
- **Im Siebener-Format** (групами по сім) — щотижневий формат

## Поширені помилки

### Помилка 1: Неправильний порядок чисел

❌ "Zwanzigeins" (неправильно)
✅ "Einundzwanzig" (правильно)

**Правило:** Німці говорять одиниці першими, потім десятки.

### Помилка 2: Неправильне утворення чисел-підлітків

❌ "Zehnsechs" (неправильно)
✅ "Sechzehn" (правильно)

**Правило:** Додавайте -zehn до числа, а не навпаки.

### Помилка 3: Забування 'und'

❌ "Einzweiunddreißig" (неправильно)
✅ "Zweiunddreißig" (правильно)

**Правило:** Використовуйте 'und' для з'єднання одиниць і десятків.

## Поради щодо вимови цифр

**Ключові звуки для освоєння:**

- **z** = [ц] як в "коти"
- **ß** = [с] гострий звук s
- **ü** = округлений [ю] як в "фюр"
- **eu** = [ой] як "бій"
- **ei** = [ай] як "ай"

## Підсумок

Німецькі цифри слідують чітким патернам:

1. **0-10** — Базові цифри для запам'ятовування
2. **11-20** — Elf, zwölf, потім цифра + zehn
3. **Десятки** — Цифра + zig (або ßig для dreißig)
4. **Складні цифри** — Одиниці + und + десятки (einundzwanzig)
5. **Сотні** — Цифра + hundert
6. **Великі цифри** — Tausend, Million, Milliarde
7. **Порядкові** — закінчення -te/-ste (erste, zwanzigste)

**Порада:** Почніть з **eins, zwei, drei, vier, fünf, zehn, zwanzig, hundert** — ці вісім цифр охоплюють більшість базових ситуацій лічби. І пам'ятайте: німці рахують інакше, ніж в англійській — одиниці першими, потім десятки (einundzwanzig = один-і-двадцять)!

Ваша німецька лексика цифр починається з лічби від одного до десяти. Viel Erfolg!
